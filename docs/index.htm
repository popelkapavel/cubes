Cubes 

Cubes is simple program for editing monochrome 3d bitmap (voxel editor) and convert it to polygon mesh object.
Mesh then can be smothed and precised (subdivided).
Then you can copy bitmap and vector (emf) to clipboard,export or save it.

Keys 'M' (mesh) or 'O' (rotoid) switch to mesh mode and then 'N' back to edit mode.


Export
                       
bitmap : png (anaglyph option)
mesh   : off,obj (wavefront),wrl,3ds
vector : emf,svg,pdf (uncompressed)

It is free for any use.
